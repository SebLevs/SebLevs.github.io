Scene assets have always been frustrating to work with

Organizing and finding them becomes increasingly bothersome as a project grows, and their innate inability to be referenced for runtime purposes makes working with scene assets a cumbersome annoyance.

Scene Manager Toolkit solves common issues with a set of tools that greatly simplifies scene asset management for both editor and runtime purposes.

Editor window
Centralized Scene Asset management

Create, rename, move into folder, delete, tag, search, and handle all states of a scene asset within a user-friendly editor window that automatically lists and keeps track of every scene asset of the project.

Tag Scene Assets

Easily tag Scene Assets for a powerful categorization that pairs with Scene Manager Toolkit's search behaviours.

Easily find Scene Assets

Multiple search behaviours offer a smooth search of scene assets, whether it is by name, by tag, or by scenes of the hierarchy.

Inspector
Frustration-free inspector reference

Scene Assets can now be referenced in the inspector like any other asset.

Buttons are dynamically shown and hidden to facilitate the build settings and addressables management.

Runtime
A straightforward code usage

SceneAssetReferences are Equatable. The GUIDs of the referenced SceneAssets are used for the comparison.


The Scene Asset can be loaded and unloaded as usual by using its cached data.


The tags linked to a Scene Asset can also be retrieved.
