This project was made within an arcade game cloning club a few colleagues and I started at the beginning of our DEC at LaSalle college.

The main goal of this project was to familiarize myself with the Unity engine while trying to replicate as closely as possible the logic of the Space Invaders game.

PROGRAMMING
ASCII and destructible buildings

Each building is made out of 1x1 pixel blocs. Each bloc is instantiated from a two-dimensional array of characters (ASCII). It surprises me to this day that I actually went through with this tedious task.

Every bloc of a building can be destroyed.
When a projectile hits a bloc, a couple of the bloc's nearest blocs are also destroyed.
